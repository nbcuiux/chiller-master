Feedback Interaction

Feedback interaction is an actions that reacts to a viewer's click or press to indicate that their click has been registered.

Contents

  • Tiles
  • Buttons
  • Text Fields

Tiles

Tiles will be used as informational containers or in a similar fashion to buttons. For the tiles that are clickable we will be associating a simple and subtle press effect that lets the user know their action has been registered with the system.

Buttons

Similarly to the tile press, the buttons will have the same press animation.

Text Fields

Text fields may be used at a later time for application forms, contest entry forms, or newsletter registrations. Continuing with the theme of the slice effect, we have created a simple slice animation once the text field has been clicked.
