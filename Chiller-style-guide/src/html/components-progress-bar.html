Progress Bar

Progress bars are useful for the viewer because it lets them know their request if being processed/loaded. On the new ChillerTV.com much of the content will be included on an infinite scroll page. These preloaders would be used to indicate to the viewer that new content is being loaded. These can also be used when loading an entirely new page as well.

Continuing to stick with the theme of the slice and the Chiller red, we have provided four variations of progress bar animations.
