Typography

Typography is the art and technique of arranging type to make written language legible, readable, and appealing when displayed. By defining the correct typeface, line height and contrast ChillTV.com will be easier to navigate, increase click through rates, and increase the overall readability and aesthetics of the website.

Typeface

Apex New

Apex New has been refined to work across a wide set of supported platforms.

Apex New font weights

Apex New has seven weights: Thin, Light, Book, Medium, Bold, Heavy, and Ultra. The slight variations in weight lends itself to creating dynamic copy and imagery for all subject matter.

Styles

Too many type sizes and styles at once can wreck any layout. A typographic scale has a limited set of type sizes that work well together along with the layout grid.

These sizes and styles were developed to balance content density and reading comfort for the average ChillerTV viewer accross all devices.

Line Height

By determining the correct line height for each style you will be able to achieve proper readability and appropriate pacing for the viewers of ChillerTV.com. It is important to not only consider the line height between text with the same style, but text with different styles as well. The correct line height will also make the information more inviting to the viewer rather than intimidating.
