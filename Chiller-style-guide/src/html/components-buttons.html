Buttons

Buttons are used to let the user know that there is a clickable action associated with an area of your website.

To reinforce the eerie and ominous feeling given off by the Chiller brand, we will be using ghost buttons for the initial state. This still gives the viewer an indication that there is a button, but at the same time does not take away from the visual that the button is placed on. As seen in the hover section, the button then will either change to a solid red background or slice with a red corner.

To view the animations for these buttons, visit the hover interactions section.
