All Shows

The shows and all movies pages should be simple to navigate and very visual. These will be two seperate pages, but they will follow the same layout.

By using a simple grid design with large tiles we are able to quickly display ChillerTV shows and movies. Using strong recognizable imagery will make the experience simplier for the viewer to navigate through the page. The tiles will use imagery from the show/movie with the show logo and short text. These tiles will then link to the invidivual page for the show/movie selected.
